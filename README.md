# GDRP Project
This project will develop a prototype for data storage on blockchain
### To run & install
1. Run local ganache instance
2. Deploy with truffle
```
truffle deploy
```
3. Install npm packages for the front end

```
cd client && npm i
```
4. Run it
```
npm start
```
